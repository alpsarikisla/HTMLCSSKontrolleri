Satır Bazında css(inline)

Etiket Seçiciler

Temelde bütün sayfa veya bir grup içerisindeki aynı etikete sahip tüm içeriklerin etkilenmesi için kullanılır. head bloğuna style etiketi tanımlanır ve seçiciler bu etiket içerisine yazılır. css özellikleri {} küme parantezi içerisine yazılarak birden çok css özelliğinin tanımlanması sağlanır.

Sınıf Seçiciler

Önce sınıf tanımlanır. body içerisinde aynı özellikte olması istenen tüm etiketlere class attribute'u ile sınıf ataması yapılır

ID Seçiciler

ID html sayfasında unique olarak atanmış veridir/etikettir. css ile bu unique elemanın özelliklerini değiştirebiliriz
